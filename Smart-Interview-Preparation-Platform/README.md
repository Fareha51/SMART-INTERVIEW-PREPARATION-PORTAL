# Smart Interview Preparation Platform

A web application designed to help students prepare for technical interviews.

## Features
- User Login
- Dashboard
- DSA Problem Tracker
- Resume Upload
- Interview Schedule

## Technologies Used
- HTML
- CSS
- JavaScript
- Local Storage

- Admin Dashboard
