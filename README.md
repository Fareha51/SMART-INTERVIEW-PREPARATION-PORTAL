# Smart Interview Preparation Platform

## Overview
Smart Interview Preparation Platform is a web application designed to help students prepare for technical interviews by tracking their DSA progress, managing resumes, and organizing interview schedules.

## Features
- User Login Interface
- Interactive Dashboard
- DSA Problem Tracker
- Resume Upload System
- Interview Schedule Management
- Responsive User Interface

## Technologies Used
- HTML5
- CSS3
- JavaScript
- Local Storage

## Future Enhancements
- Spring Boot Backend
- MySQL Database Integration
- User Authentication
- Multi-user Support
- Admin Dashboard
- Progress Analytics
